correrMovimentos :: Jogo -> [Movimento] -> Jogo #

A função correrMovimentos é a função principal desta tarefa sendo ela responsável por fazer acontecer os movimentos dados através do input do utilizador. Nela estão presentes várias funções auxiliares as quais serão explicadas posteriormente.

moveJogador :: Jogo -> Movimento -> Jogo #

Esta função tem como objetivo fazer movimentar, realizar as ações relativas à caixa, seja pegando nela ou largando-a (sendo que damos o valor False quando o jogador não possui uma Caixa, e o valor True quando o jogador possui uma Caixa), e também a ação de trepaObjeto seja um Bloco ou uma Caixa

coordPeca :: Mapa -> Coordenadas -> (Int, Int) -> Peca #

A função coordPeca serve para facilitar praticamente todas as outras funções já que esta transmite a informação de qual a peça numa Coordenada específica.

avancar :: Jogo -> Movimento -> Jogo #

A função avancar corresponde à atualização do estado do Jogo quando ocorre o Jogador avança.

movimentar :: Jogo -> Movimento -> Coordenadas -> Jogo #

A função movimentar corresponde à atualização do estado do Jogo quando ocorre o Jogador se movimenta.

dropCaixa :: Jogo -> Coordenadas -> Jogo #

A função dropCaixa tem como objetivo primordial largar a Caixa portada pelo jogador, nas diferentes direções possíveis, quer seja para a Esquerda (Oeste) ou para a Direita (Este).

moveCaixa :: Jogo -> Jogo #

A função moveCaixa serve para tal como diz o seu nome para mover a caixa, e para tal, tomamos a opção de enumerar todas as diferentes ocasiões onde tal ação pode acontecer.

trocaCaixaVazio :: [Peca] -> Int -> Int -> [Peca] #

Esta função aplica uma troca entre uma caixa e um vazio substituindo, uma Peça Caixa por um espaço vazio.

removeVazio :: Mapa -> Coordenadas -> Int -> Mapa #

removeVazio é uma função que utiliza como auxiliar a função anterior, e a sua utilidade é remover um espaço vazio específico do mapa.

trocaVazioCaixa :: [Peca] -> Int -> Int -> [Peca] #

Esta função aplica uma troca entre um espaço vazio e uma caixa substituindo, um espaço vazio por uma Peça Caixa.

removeCaixa :: Mapa -> Coordenadas -> Int -> Mapa #

Tal como a removeVazio, a removeCaixa é uma função que tem como utilidade fundamental remover uma peça Caixa do mapa utilizando como auxiliar a função anterior trocaVazioCaixa.
